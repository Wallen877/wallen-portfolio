## Run locally

1. Clone the repo:

```bash
git clone https://github.com/Wallen877/wallen-portfolio.git
cd wallen-portfolio
```

2. Open `index.html` in your browser or run a simple local server:

```powershell
# Windows: preview in default browser
Start-Process .\index.html

# or run a simple static server with Python
python -m http.server 8000
# then open http://localhost:8000
```
